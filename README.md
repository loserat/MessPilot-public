# Öffentliches Support- & Release-Repository

Dieses Repository ist die öffentliche Anlaufstelle für Informationen, Release-Notizen, Fehlerberichte, Funktionswünsche und allgemeine Diskussionen zur Software.

Wichtig: Der Quellcode der Anwendung ist nicht öffentlich in diesem Repository enthalten. Dieses Repository enthält keine produktiven App-Quellcodes, keine sensiblen Daten und keine internen oder kundenspezifischen Informationen.

## Fehler melden

- Bitte nutzen Sie die Issue-Vorlage `Bug Report` für reproduzierbare Fehlerberichte.
- Geben Sie möglichst genaue und reproduzierbare Schritte an sowie die verwendete Version und ggf. Browser/Gerät.
- Posten Sie keine sensiblen Daten, keine personenbezogenen Kundendaten oder internen Namen.

## Funktionswünsche

- Für neue Ideen oder Verbesserungen verwenden Sie die Vorlage `Feature Request`.
- Beschreiben Sie Problem, gewünschte Lösung und den erwarteten Nutzen.

## Diskussionen / Fragen

- Für allgemeine Fragen, Erfahrungsaustausch oder Diskussionen nutzen Sie bitte die GitHub Discussions.

## Releases

- Release-Informationen und Änderungen werden im `CHANGELOG.md` gepflegt.

## Roadmap

- Die öffentliche Roadmap finden Sie in `ROADMAP.md`. Dort sind geplante Themen, in Prüfung befindliche Punkte und später vorgesehene Erweiterungen gelistet.

## Sicherheit / Datenschutz

- Sicherheitsprobleme bitte nicht als öffentliches Issue melden. Nutzen Sie den privaten Kontaktweg (siehe `SECURITY.md`).
- Bitte geben Sie in Issues oder Discussions keine personenbezogenen oder sensiblen Daten preis.
