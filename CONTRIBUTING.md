# Mitwirken / Contributing

Vielen Dank für Ihr Interesse, das Projekt zu unterstützen. Bitte beachten Sie die folgenden einfachen Regeln, damit Beiträge für alle hilfreich bleiben.

## Grundregeln
- Bleiben Sie sachlich und respektvoll.
- Vermeiden Sie personenbezogene Daten und interne Firmendaten.

## Fehlermeldungen (Bugs)
- Nutzen Sie die `Bug Report`-Vorlage.
- Liefern Sie reproduzierbare Schritte, die verwendete Version und Angaben zu Browser/Gerät wenn relevant.
- Hängen Sie nur nicht-sensible Logs/Screenshots an. Entfernen Sie vorher alle personenbezogenen Daten.

## Funktionswünsche
- Nutzen Sie die `Feature Request`-Vorlage.
- Beschreiben Sie Problem, gewünschte Lösung, Nutzen und ggf. Priorität.

## Diskussionen
- Nutzen Sie die Discussions für allgemeine Fragen oder Ideen.

## Datenschutz und sensible Inhalte
- Posten Sie keine Kundendaten, keine echten Namen, keine internen Projektbezeichnungen oder sensiblen Geschäftsinformationen.

---

Wenn Sie unsicher sind, kontaktieren Sie bitte die maintainers über die vorgesehenen Kanäle in diesem Repository.
