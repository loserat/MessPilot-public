# MessPilot

Dieses Repository dient als öffentliche Anlaufstelle für MessPilot.

Hier findest du:

- Release Notes
- Fehlerberichte
- Funktionswünsche
- Diskussionen
- Roadmap-Informationen

Der eigentliche Quellcode befindet sich in einem privaten Repository.

## Fehler melden

Bitte nutze den Bereich **Issues**, um Fehler zu melden.

## Fragen und Ideen

Bitte nutze **Discussions** für Fragen, Ideen und allgemeines Feedback.

## Releases

Öffentliche Versionshinweise findest du unter **Releases**.
